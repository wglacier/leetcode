### 50. Pow(x, n)
Implement pow(x, n), which calculates x raised to the power n (xn).

Example 1:

Input: 2.00000, 10
Output: 1024.00000
Example 2:

Input: 2.10000, 3
Output: 9.26100
Example 3:

Input: 2.00000, -2
Output: 0.25000
Explanation: 2-2 = 1/22 = 1/4 = 0.25
Note:

-100.0 < x < 100.0
n is a 32-bit signed integer, within the range [−231, 231 − 1]
```c++
class Solution {
public:
    double myPow(double x, int n) {
        if (n == 0) return 1.0;
        
        if (n < 0) {
            if (n == INT_MIN) {
                return 1.0/(myPow(x, INT_MAX)*x);
            }
            return 1.0/myPow(x, -n);
        }
        long i = 1;
        double res = x;
        while (i*2 <= n) {
            res = res * res;
            i = i << 1;
        }
        return res * myPow(x, n - i);
    }
};
```
